Fjango Fragments by mariocesar

A simplistic django app to share snippets and pastes with others. All code fragments are Mercurial repositories, so they are automatically versioned, forkable and usable as a Mercurial repository.

Dependencies

* Django 1.3 * Mercurial >= 1.9 * python-pygments

Authors

Mario César Señoranis Ayala

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/mariocesar/django-fragments

get the source code on GitHub : mariocesar/django-fragments
